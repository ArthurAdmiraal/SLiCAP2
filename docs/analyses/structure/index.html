Structure

Imagine you want to know the following information of your circuit: the transfer function from your circuit's input port (let's call this port port_in) at your circuit's output port (let's call this port port_out) in the Laplace domain.

In SLiCAP, you can obtain this information using the following command.

my_circuit.transfer().from('port_in').at('port_out').laplace

This command consists of 4 different pieces of information:

  1. What circuit you want to analyse (the circuit object my_circuit)
  2. What type of information you want to know (a transfer function)
  3. Details about the exact information you want to know (in the case of a transfer function: the ports between which you want to know the transfer)
  4. How the output should be represented (in the Laplace domain)

All core analysis commands share this same structure!
