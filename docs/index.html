Introduction

Overview#

Circuit analysis with SLiCAP usually starts with a schematic, which can be drawn using the specialised SLiCAP libraries in supported CAD tools. For example, consider this opamp noninverting amplifier in LTSpice.

The schematic is loaded into a Schematic object for analysis.

amplifier = Schematic.load('amplifier.asc')

Schematics can be drawn directly from SLiCAP to check that they were imported directly.

amplifier.draw()

Supported analyses can be performed on schematics. For example, the gain of the amplifier can be found through the following command.

amplifier.transfer().from('Vs').at('Out').laplace
